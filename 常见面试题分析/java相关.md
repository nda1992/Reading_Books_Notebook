#### 类加载的过程？

类的加载过程为：装载、链接、初始化.<br>

java中的类加载器：

- Bootstrap Loader
- ExtClassLoader
- AppClassLoader

什么时候会触发类的初始化：<br>

- 创建类的实例（new 一个对象时）
- 访问某个类或接口的静态变量，（或对该静态变量赋值）
- 调用类的静态方法
- 反射
- 初始化一个类的子类
- JVM启动时标明的启动类
